# Инструкция по развертыванию проекта
**Все команды выполнять из корневой папки проекта!**
## Backend
### Установка зависимостей 
```
cd backend
npm install
```
### Запуск
```
cd backend
node server.js
```

Сервер запустится на `http://localhost:3000`
## Frontend

### Установка зависимостей 
```
cd frontend
npm install
```

### Запуск
```
cd frontend
node server.js
```
Сервер запустится на `http://localhost:3001`
